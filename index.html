O que é o Projeto Baja SAE?

O Baja SAE é uma competição estudantil organizada pela SAE Brasil onde equipes universitárias projetam, constroem e competem com um protótipo de carro off-road. O projeto exige aplicação real de conhecimentos de engenharia, trabalho em equipe e inovação, passando por etapas como aceleração, frenagem, tração, e resistência.

Sobre o Projeto

Este é um projeto acadêmico fictício desenvolvido por estudantes de Análise e Desenvolvimento de Sistemas do UniCuritiba como parte de um trabalho de disciplina. O objetivo principal é demonstrar conhecimentos em:

  • Desenvolvimento web com HTML, CSS e JavaScript
  • Aplicação de cálculos técnicos automotivos
  • Design de interface focado em usabilidade
  • Compreensão dos fundamentos da competição Baja SAE

Importante: Este site não está vinculado a nenhuma equipe real de Baja SAE. É uma simulação prática criada exclusivamente para fins educacionais e avaliação acadêmica.
